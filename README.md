Emissor de Certificados - IFCE Boa Viagem

Sistema web simples para geração de certificados individuais, desenvolvido para apoiar a disciplina de Projeto Integrador do Instituto Federal do Ceará (IFCE) – Campus Boa Viagem.

Objetivo

Permitir que professores gerem certificados personalizados de forma rápida, automática e padronizada para alunos participantes de projetos acadêmicos.

Funcionalidades
✅ Cadastro de dados do projeto:
Nome do sistema desenvolvido
Nome do orientador(a)
Cliente externo
Semestre
✅ Cadastro dinâmico de alunos:
Nome
Matrícula
✅ Geração automática de certificados individuais
✅ Layout pronto para impressão (formato A4 horizontal)
✅ Impressão direta pelo navegador
🖥️ Tecnologias Utilizadas
HTML5
CSS3
JavaScript (Vanilla JS)
Estrutura do Projeto
certificados-ifce/
│── index.html
│── imagelogo.png
▶️ Como Executar o Projeto
Baixe ou clone este repositório:
git clone https://github.com/SEU-USUARIO/certificados-ifce.git
Abra o arquivo index.html no navegador:
cd certificados-ifce
Clique duas vezes no arquivo index.html
ou
Clique com botão direito → "Abrir com navegador"

Como Usar
Preencha os dados do projeto
Adicione os alunos
Clique em "Gerar Certificados Individuais"
O navegador abrirá automaticamente a tela de impressão
Escolha:
Imprimir
Salvar como PDF
⚠️ Observações
Certifique-se de que a imagem imagelogo.png esteja na mesma pasta do index.html
Necessário navegador atualizado (Chrome, Edge, Firefox)
